** Implemented Options

*** Sidebar
- ~layout~ - "tabbed" | "stacked" | "favorites_only" | "drives_only"
- ~tabStyle~ - "manila" | "underline" | "pill"

*** FileList
- ~showDotfiles~ - Show/hide dotfiles
- ~sortAscending~ - Sort order

*** Search
- ~engine~ - "builtin" | "ripgrep" | "ugrep"
- ~defaultDepth~ - Search depth limit

*** Preview
- ~enabled~ - Show preview pane
- ~widthPercent~ - Preview pane width
- ~textExtensions~ - File types for text preview
- ~imageExtensions~ - File types for image preview
- ~maxFileSize~ - Max file size to preview
- ~markdownRendered~ - Render markdown or show raw

** Implementation Priority Suggestions

*** High Priority (Commonly Expected Features)
1. ~behavior.confirmDelete~ - Add confirmation dialog before delete
2. ~behavior.restoreLastPath~ - Remember last directory on startup
3. ~tabs.restoreTabsOnStart~ - Remember tabs between sessions

*** Medium Priority (Nice to Have)
1. ~sidebar.width~ - Configurable sidebar width
2. ~preview.position~ - Preview on right/bottom
3. ~fileList.rowHeight~ - Compact/normal/comfortable modes
4. ~tabs.lastTabBehavior~ - What to do when closing last tab

*** Low Priority (Power User Features)
1. ~sidebar.collapsible~ - Collapsible sidebar
2. ~toolbar.visible~ - Hide toolbar option
3. ~search.rememberLastQuery~ - Persist search queries
4. ~statusBar.visible~ - Hide status bar option
